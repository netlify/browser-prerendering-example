This page demonstrates how to use <meta name="prerender-status-code"> to control the HTTP status code returned by the prerender function.

Meta tag in <head>:
<meta name="prerender-status-code" content="404">

When the prerender function processes this page, it:

  1. Renders the HTML content normally
  2. Looks for <meta name="prerender-status-code" content="404">
  3. Returns this HTML with a 404 status code instead of 200
  4. Sets appropriate cache headers for error pages

Content Details

Even though this returns a 404 status code, it can still contain rich HTML content for SEO purposes.
